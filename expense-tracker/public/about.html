🛠️ Tech Stack

Backend

  • ✅ Node.js (v18+) — Runtime environment
  • ✅ Express.js — Web framework & routing
  • ✅ REST API — CRUD endpoints
  • ✅ CORS middleware — Cross-origin support
  • ✅ dotenv — Environment configuration
  • ✅ uuid — Unique ID generation
  • ✅ In-memory storage (no DB required)

Frontend

  • ✅ HTML5 — Semantic markup
  • ✅ Tailwind CSS (CDN) — Responsive styling
  • ✅ Vanilla JavaScript — Frontend logic
  • ✅ Chart.js — Data visualizations
  • ✅ Fetch API — Backend communication
  • ✅ Custom CSS animations
  • ✅ Google Fonts — Typography

🚀 Deployment Steps (Render.com)

  1. 1Push project to GitHub repository
  2. 2Create account at render.com
  3. 3Click "New +" → "Web Service" → Connect GitHub repo
  4. 4Set Build Command: npm install
  5. 5Set Start Command: node server.js
  6. 6Add environment variable: NODE_ENV=production
  7. 7Click "Create Web Service" — deployment begins automatically!
